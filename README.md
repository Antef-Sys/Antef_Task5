# 🧺 Laundry Wallah
https://antef-sys.github.io/Antef_Task5/

## 📌 Project Overview

Laundry Wallah is a simple and clean frontend web project designed to showcase a modern laundry service landing page. It highlights services, branding, and a call-to-action for users to book laundry services easily.

---

## 🚀 Features

* 🧼 Attractive landing page design
* 🧭 Navigation bar with logo, menu, and user section
* 📝 Service description and heading section
* 🖼️ Image section for visual appeal
* 🔘 Call-to-action button (Book Service)
* 🎨 Clean UI with consistent color scheme

---

## 🛠️ Technologies Used

* HTML5
* CSS3
